Política de Privacidade - Luiza Art Nails

Esta Política de Privacidade descreve como o aplicativo Luiza Art Nails coleta, usa e protege os dados dos usuários.

1. Dados coletados

O aplicativo pode coletar dados fornecidos pelo usuário, como nome, e-mail, telefone e informações de agendamento.

2. Finalidade do uso

Os dados são utilizados para permitir login, cadastro, gerenciamento de agendamentos e funcionamento geral do aplicativo.

3. Armazenamento e segurança

Os dados podem ser armazenados em serviços de terceiros utilizados pelo app, como Firebase, com medidas razoáveis de segurança para proteção das informações.

4. Compartilhamento de dados

Os dados não são vendidos. Eles podem ser processados por provedores essenciais para o funcionamento do aplicativo.

5. Exclusão de dados

O usuário pode solicitar a exclusão de sua conta e de seus dados entrando em contato pelo e-mail informado abaixo.
